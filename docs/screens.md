## 1. ダッシュボード

### 上部：主要KPIカード

| カード | 内容 |
|--------|------|
| アクティブユーザー数 | 今月 / 先月比 |
| 離反予備軍数 | 条件カスタマイズ可 |
| 新規顧客数 | 今月 / 先月比 |
| 平均客単価 | 今月 / 先月比 |
| 平均利用パートナー数 | 今月 / 先月比 |
| 新規利用パートナー数 | 今月 / 先月比 |

### 中部：グラフ

| グラフ | 内容 |
|--------|------|
| ログイン推移 | 日別アクティブユーザー数の折れ線 |
| 注文推移 | 日別注文件数・金額の折れ線 |
| 離反率推移 | 月別の離反率トレンド |

### 下部：アクションリスト

| リスト | 内容 |
|--------|------|
| 要対応ユーザー | 離反リスク高・カゴ落ち等、今すぐアクションすべきユーザー |
| 直近の配信結果 | 最新の配信とその開封率・反応サマリ |

### 離反/新規分析

ダッシュボード上ではサマリ表示。クリックで詳細画面に遷移。

| 項目 | 仕様 |
|------|------|
| 比較期間 | 自由選択（先週vs今週、先月vs今月、任意期間） |
| 切り口 | 顧客別 → パートナー別 → 商品別（3階層） |
| 基準 | 注文回数・注文金額・注文頻度（選択 & 組み合わせ可） |

**離反分析**: 以前は買っていたが、最近買わなくなった顧客・パートナー・商品を検知
**新規分析**: 以前は買っていなかったが、新たに買い始めた顧客・パートナー・商品を検知

## 4. セグメント作成 + 行動検索

### 概要
ユーザー属性と行動データを組み合わせて、ターゲットグループを作成・保存できる画面。

### 属性条件

| 条件 | 例 |
|------|-----|
| 顧客コード | 指定 / 範囲 |
| エリア | 住所で絞り込み |
| ステータス | アクティブ / 離反リスク / 離反 / 新規 |
| 利用パートナー | 特定パートナーの利用有無 |
| LINE連携 | 連携済み / 未連携 |

### 行動条件

| 条件 | 例 |
|------|-----|
| ログイン | 直近N日以内 / N日以上前 |
| ページ閲覧 | 特定ページを見た / 見ていない |
| サイト内検索 | 特定キーワードで検索した |
| 注文 | 金額・回数・頻度の範囲指定 |
| カゴ落ち | カゴ落ちあり / なし |
| 流入元 | LINE経由 / Google検索 etc. |

### 条件の組み合わせ
- AND / OR で複数条件を組み合わせ可能
- 例：「最終ログイン30日以上前」AND「先月注文5万円以上」→ 高単価離反リスク

### 結果エリア
- 条件に合致するユーザー数をリアルタイム表示
- ユーザー一覧をプレビュー表示
- セグメント名をつけて保存

### セグメントの種類

| 種類 | 仕組み | 用途 |
|------|--------|------|
| 動的セグメント | 条件を保存。毎日自動で対象者が更新される | 自動配信ルール向き |
| 静的セグメント | 作成時点のユーザーリストを固定 | キャンペーン等の1回限りの配信向き |

保存時に「動的 / 静的」を選択可能。

### アクション
- 保存したセグメントに対して → そのまま手動配信画面へ
- 保存したセグメントを → 自動配信ルールの対象に設定

## 5. 手動配信

### 配信フロー

```
① 対象を選ぶ → ② チャネルを選ぶ → ③ メッセージを作る → ④ 確認 → ⑤ 送信
```

### ① 対象選択

| 方法 | 内容 |
|------|------|
| セグメントから | 保存済みセグメントを選択 |
| ユーザー一覧から | チェックボックスで選択した状態から遷移 |
| 個別指定 | ユーザー詳細画面から遷移 |

### ② チャネル選択

| チャネル | 条件 |
|---------|------|
| LINE | LINE連携済みユーザーのみ |
| メール | メールアドレスありのユーザーのみ |
| SMS | 電話番号ありのユーザーのみ |

- 単一チャネル選択可
- 複数チャネル同時送信も可（LINE + メール等）

### ③ メッセージ作成
- テキスト入力
- テンプレートから選択（Phase 2で実装）
- 差し込み変数（顧客名、店舗名 etc.）

### ④ 確認画面
- 送信対象人数
- メッセージプレビュー
- 送信チャネル

### ⑤ 送信
- 即時送信
- 予約送信（日時指定）

## 6. 自動配信ルール設定

### ルール設定フロー

```
① トリガー条件を設定 → ② アクションを設定 → ③ 有効化
```

### ① トリガー条件

| カテゴリ | 例 |
|---------|-----|
| ログイン | 最終ログインからN日経過 |
| 注文 | 注文頻度がN%低下 / N日間注文なし |
| カゴ落ち | カートに入れたがN時間以内に未注文 |
| ページ閲覧 | 特定ページを閲覧した |
| 検索 | 特定キーワードで検索した |
| セグメント | 動的セグメントに新しく入った / 出た |

### ② アクション

| 設定項目 | 内容 |
|---------|------|
| チャネル | LINE / メール / SMS（単一 or 複数選択可） |
| メッセージ | テキスト入力 + 差し込み変数 |
| 送信タイミング | 即時 / N時間後 / N日後 |
| サイト内バナー表示 | Phase 2で実装 |

### ③ ルール管理

| 機能 | 内容 |
|------|------|
| 有効 / 無効 | ワンクリックで切り替え |
| 実行履歴 | いつ誰に送ったかのログ |
| 重複防止 | 同じユーザーには最低N日間空ける（N日は設定可能） |

※ 自動配信の上限数は設けない

## 7. 配信履歴

### 一覧表示

| 項目 | 内容 |
|------|------|
| 配信日時 | いつ送ったか |
| 配信種別 | 手動 / 自動 |
| チャネル | LINE / メール / SMS |
| 対象 | セグメント名 or 個別ユーザー名 |
| 送信数 | 何人に送ったか |
| 開封率 | 開封した人の割合 |
| クリック率 | リンクをクリックした人の割合 |

### 詳細画面（個別配信をクリック）

| 項目 | 内容 |
|------|------|
| メッセージ内容 | 送ったテキスト・画像 |
| 送信先一覧 | 誰に送ったか（開封/未開封のステータス付き） |

### 効果測定指標

| 指標 | 内容 |
|------|------|
| 開封率 | メッセージを開いた割合 |
| クリック率 | リンクをクリックした割合 |
| 配信後N日以内の注文率 | 送った人のうち何%が注文したか |
| 配信後N日以内の注文金額 | 配信対象者の合計注文額 |
| 配信後のログイン率 | 送った人のうち何%がログインしたか |
| ブロック/配信停止率 | LINEブロック・メール配信停止の割合 |
| 未配信者との比較 | 送った人 vs 送らなかった人の注文率差 |

### フィルタ

| フィルタ | 内容 |
|---------|------|
| 期間 | 日付範囲指定 |
| チャネル | LINE / メール / SMS |
| 種別 | 手動 / 自動 |
| ルール名 | 自動配信のルール別 |
