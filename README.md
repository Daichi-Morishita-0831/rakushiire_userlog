# rakushiire-crm

rakushiire.com ユーザー行動管理 & コミュニケーションツール

## 概要

rakushiire.com のユーザー行動データを日次で収集・可視化し、CRMチームが手動/自動でコミュニケーション（LINE・メール・SMS）を送れる管理ツール。
